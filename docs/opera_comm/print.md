# 打印

1. 一页A4放多张ppt
    - 打印处理选择“每张纸上放置多页”，根据需要选x*x的分布
    - 纸张方向选择横向
    - 横向后记得选短边翻转，否则正面是正的字，反面是反的字

2. pdf拆分
    - 福昕PDF，页面，右键选择拆分文档

3. caj转pdf
    - 一般情况下直接打印，选择print to pdf
    - 如果caj加密了，一次只能打2页，还要合并，很麻烦
      - 用caj7.2 精简绿色版（在D盘已经安装）可以一次性转成pdf  
      - 转之后生成ori那个高清体积大的
      - 可以利用wps压缩成比较小的版本
